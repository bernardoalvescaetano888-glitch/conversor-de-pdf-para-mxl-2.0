Converta seus arquivos PDF para XML para abri-los em seu aplicativo de preferência. Acesse o link abaixo para baixar os arquivos necessários:

https://drive.google.com/drive/folders/155gVYJAFJePpE_R-l3qg-2lBrbhLLDXC?usp=drive_link

Instruções de instalação:

Baixe o Python 3.14.

Baixe o arquivo MSI disponível na pasta do repositório no Drive.

Após a instalação, abra o prompt de comando (CMD).

Digite cd seguido do caminho da pasta onde você baixou os arquivos.

Execute o comando: pip install customtkinter.

Após concluir esses passos, basta rodar o programa, selecionar seu PDF e realizar a conversão em minutos, de forma totalmente gratuita!
